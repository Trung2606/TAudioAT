Cài đặt

    1. Cài môi trường python

  • Cài đặt python với câu lệnh:

    sudo apt-get install python3

  • Cài đặt pip với câu lệnh:

    sudo apt-get install python3-pip

  • Cài đặt tkthemes với câu lệnh:

    pip install ttkthemes

  • Cài đặt pygame với câu lệnh:

    pip3 install pygame

  • Cài đặt mutagen với câu lệnh:

    pip3 install mutagen
