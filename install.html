Cài đặt TAudioAT

TAudioAT là một phần mềm nghe nhạc được viết bằng ngôn ngữ lập trình python trên hệ điều hành dành cho Linux(Ubuntu). Sau đây ta sẽ tiến hành cài đặt môi trường cần thiết cho phần mềm TAudioAT.

Cài đặt python:

Khởi động Terminal và thực thi dòng lệnh sau để tiến hành cài đặt Python:

sudo apt-get install python3

Cài đặt pip:

Tiếp tục thực hiện câu lệnh sau để cài đặt pip:

sudo apt-get install pip3

Cài thư viện ttkthemes:

Tiếp tục thực hiện câi lệnh sau để cài thư viện ttkthemes:

pip install ttkthemes

Cài thư viện pygame:

Tiếp tục thực hiện câi lệnh sau để cài thư viện pygame:

pip3 install pygame

Cài thư viện mutagen:

Tiếp tục thực hiện câi lệnh sau để cài thư viện mutagen:

pip3 install mutagen
